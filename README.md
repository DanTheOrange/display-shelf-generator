## Display Shelf Tool

Tool for planning nested display shelves cut from plywood sheets. It generates:
- Plywood sheet layouts with labeled parts.
- A 3D preview of the assembled shelf stack.

### Stack
- React + Vite + TypeScript
- Tailwind + shadcn/ui
- Three.js + React Three Fiber + drei

### Getting Started
```bash
pnpm install
pnpm dev
```

### Notes
- Units are metric (mm).
- Parts are auto-laid out with no manual editing.
- Use the sidebar to configure plywood size, shelf dimensions, tool gap, and notch settings.
